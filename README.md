# Weather Information - Current conditions and forecasts

**Package**: `@system/weather`
**Version**: 0.1.0
**Repository**: `pkg-system-weather`

## Overview

Weather Information - Current conditions and forecasts

## Installation

### Prerequisites

This package requires the following external commands:

- `curl`
- `jq`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @system/weather

# Or install from local source
cd ~/.pkgs
stow weather
```

## Usage

### System Integration

This package provides system integration for weather information - current conditions and forecasts.


## Configuration

Configuration files are typically stored in:
- `~/.config/weather/` - User configuration
- `~/.local/share/weather/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @system/weather

# Or manual unstow
cd ~/.pkgs
stow -D weather
```
